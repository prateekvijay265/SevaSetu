# SevaSetu Foundation 🌿
**Bridging Service and Social Impact**

An editorial, human-centered web application built for **SevaSetu Foundation**, an organization dedicated to empowering communities across India through education, women empowerment, environmental sustainability, and community welfare since 2012.

---

## ✨ Design Philosophy & Aesthetics

Designed with a warm, natural, and lifelike editorial aesthetic inspired by high-impact organizations (`Patagonia`, `Charity: Water`):
- **Warm Parchment & Cream Palette**: Built upon organic paper-like textures (`#FAF8F5`, `#1D6A45`, `#C0533A`, `#B5722A`) to evoke trust, warmth, and human connection.
- **Editorial Typography**: A refined pairing of **Playfair Display** (Serif) for emotional storytelling, **DM Sans** for clean accessibility, and **DM Mono** for precise data visualization.
- **Dynamic Micro-Interactions**: Smooth, spring-based transitions powered by `Framer Motion` for intuitive, tactile engagement without heavy, artificial glow effects.

---

## 🏗️ Six Core Pillars

1. **📚 Education**: Bridging the rural-urban literacy gap through accessible e-learning centers and scholarship programs.
2. **👩‍🎓 Women Empowerment**: Fostering financial independence through skill development, microfinance, and leadership training.
3. **🌱 Environmental Sustainability**: Afforestation, clean water initiatives, and renewable energy adoption across rural communities.
4. **🐾 Animal Welfare**: Compassionate care, shelter support, and rehabilitation programs for urban and rural wildlife.
5. **🤝 Community Welfare**: Grassroots healthcare camps, disaster relief mobilization, and nutrition drives across 30+ states.
6. **🚀 Youth Development**: Mentorship, vocational training, and career guidance for young change-makers.

---

## 🛠️ Technology Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router & Turbopack)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: Vanilla CSS (`globals.css`) with custom design tokens, modern grid layouts, and zero external utility bloating.
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **3D & Visualizations**: [Three.js](https://threejs.org/) / React Three Fiber

## 🚀 Getting Started Locally

### 1. Prerequisites
Ensure you have **Node.js 18+** installed on your system.

### 2. Installation
Clone the repository and install dependencies:

```bash
git clone https://github.com/yourusername/sevasetu-website.git
cd sevasetu-website
npm install
```

### 3. Run Development Server
Launch the local Next.js server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to explore the experience.

---

## 📦 Production Build & Deployment

To verify the build or compile for production:

```bash
npm run build
npm start
```

### Deploying to Vercel
The easiest way to deploy is directly via Vercel:

```bash
npx vercel
```
